Create a React TypeScript frontend for the CA marketplace with:

1. Project structure:
   - src/
     - components/ (reusable components)
     - pages/ (main pages)
     - services/ (API calls)
     - store/ (Redux state)
     - utils/ (helpers)
     - App.tsx, index.tsx

2. Pages needed:
   - Home/Landing page
   - Login/Register pages (with role selection)
   - Client Dashboard
   - CA Dashboard
   - CA Listing/Browsing page
   - Service Request details
   - Profile page

3. Libraries to use:
   - React Router for navigation
   - Redux Toolkit for state management
   - Axios for API calls
   - Tailwind CSS for styling
   - React Hook Form for forms
   - React Query (optional, for data fetching)

4. Setup:
   - Create React App with TypeScript template
   - Configure Tailwind CSS
   - Set up Redux store
   - Create API service with interceptors for JWT
